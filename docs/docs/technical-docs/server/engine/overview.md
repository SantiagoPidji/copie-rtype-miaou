# R-TYPE Engine Overview

The R-TYPE engine is a custom **C++20** game engine designed for modularity, performance, and reusability.  
It powers the client and server logic, handling game objects, rendering, physics, audio, and AI.

---

## Key Components

- **ECS (Entity Component System)**: Modular architecture for game objects.
- **Rendering**: Handles graphics, sprites, and animations.
- **Audio**: Plays sound effects and background music.
- **Physics**: Collision detection, movement, and interactions.
- **AI**: Enemy behaviors and game logic automation.
